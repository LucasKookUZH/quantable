## How to install:
for CRAN version

```r
install.packages("quantable")
```

for development version from github

```r
install.packages("devtools")
library(devtools)
install_github("wolski/quantable")
```

### for developers

downlod git repo. Use roxygenize2 to document new funcitons

```r
library(devtools)
document()
```
